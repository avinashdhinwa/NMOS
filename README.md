# NMOS
An OS I'm working on.

## Compiling ##

To compile you will need a <a href="http://wiki.osdev.org/GCC_Cross-Compiler">i686 cross-compiler</a>

### All files: ###

	make

or

	make all

### Specific file: ###

	make foo.c

Replace foo.c with the name of the file.

### Make and run: ###

	make run
